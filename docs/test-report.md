## 一、测试方法与工具

所有测试通过 cURL 命令行直接调用 REST API，验证真实 HTTP 请求/响应：

```bash
# 示例：文本问答测试命令
curl -X POST http://localhost:8080/ask/text \
  -H "Content-Type: application/json" \
  -d '{"text":"你好","stream":false}'

# 示例：语音问答测试命令
curl -X POST http://localhost:8080/ask/voice \
  -F "audio=@docs/test_voice.wav" -F "voice_output=false"
```

每条用例记录完整的「请求内容 → 预期结果 → 实际输出」，对比验证。
