## 测试用例

### 用例1：新用户文本问答 + CRM 自动注册

**输入：**
```
POST /ask/text
{"text": "你好，我叫赵六，我买了旗舰套餐，想知道有没有课程可以学", "user_id": "13600004444", "stream": false}
```

**预期结果：**
- 系统自动创建用户赵六
- RAG 检索课程信息（场景联动大师班、HA深度定制）
- 结合用户画像（旗舰套餐用户）个性化推荐

**实际结果：** ✅ 通过
```
回复摘要: "赵六哥直接上旗舰套餐，大手笔欸！你买了套餐课程直接打9折...
场景联动大师班 ¥1,299，HA深度定制 ¥2,499..."
```

**校验项：**
- [x] 新用户自动注册（`POST /crm/users/13600004444` 返回 name=赵六）
- [x] LLM 画像提取（interests=['课程学习'], facts={购买的套餐: '旗舰套餐'}）
- [x] 对话记录已保存（conversations 表有记录）
- [x] 情感追踪（sentiment_history 有记录）

### 用例2：老用户多轮对话 + CRM 闭环

**第1轮输入：**
```
POST /ask/text
{"text": "我的旗舰套餐保修到什么时候？", "user_id": "13800001111", "stream": false}
```

**第1轮实际结果：** ✅ 通过
```
回复: "张三哥，你的旗舰套餐保修到2027年3月15日，还稳稳在保修期内。
延保一年只要¥399，VIP身份不续一下太可惜了吧？"
```
- 识别用户身份（张三，VIP客户）
- RAG 检索到旗舰套餐保修期2年
- 计算到期日 = 购买日2025-03-15 + 2年 = 2027-03-15

**第2轮输入：**
```
POST /ask/text
{"text": "之前提到的那个场景联动课程多少钱？", "user_id": "13800001111", "stream": false}
```

**第2轮实际结果：** ✅ 通过
```
回复: "场景联动大师班 ¥1,299，16小时8次课，你有套餐还能打9折..."
```
- 上下文关联正确（"之前提到的" → 正确理解为场景联动课程）
- CRM 画像注入（识别为老用户，不需要重复自我介绍）

**校验项：**
- [x] 多轮对话上下文保持
- [x] CRM topic_stats 递增（function_call: 3→4）
- [x] 对话记录数递增（conversations: 2→3→4）
- [x] 用户画像持续更新
